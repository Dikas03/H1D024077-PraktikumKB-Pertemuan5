# Sistem Pakar Diagnosa Penyakit THT

Repository ini berisi source code aplikasi **Sistem Pakar Diagnosa Penyakit THT** berbasis *console* yang ditulis menggunakan bahasa pemrograman Python. Project ini dibuat untuk memenuhi tugas mata kuliah Praktikum Kecerdasan Buatan (KB) / Sistem Pakar 2 pada Pertemuan 5.

## Deskripsi Program

Aplikasi ini menggunakan metode inferensi *Forward Chaining* sederhana untuk mendiagnosis penyakit THT berdasarkan gejala yang dialami oleh pengguna. Terdapat **37 gejala** dan **23 jenis penyakit THT** yang telah dipetakan di dalam basis pengetahuan (database) program sesuai dengan studi kasus pada modul praktikum.

Daftar penyakit yang dapat didiagnosa antara lain:
* Tonsilitis
* Sinusitis (Maksilaris, Frontalis, Edmoidalis, Sfenoidalis)
* Faringitis & Laringitis
* Otitis Media Akut
* Kanker (Laring, Leher & Kepala, Nafasoring, Tonsil)
* Meniere, Vertigo, dan penyakit THT lainnya.
